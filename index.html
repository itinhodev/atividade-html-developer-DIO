Estrutura básica do HTML

A estrutura básica do HTML comporta coisas importantes, dentro dela usamos por exemplo o head e o body , onde o head comportam as coisas que vem na "cabeça" do site e o body comporta todo o conteúdo do corpo do html como titulos, immagens, textos, vídeos e etc.

Listas

As tags de lista são separadas por OL e UL, onde OL, significa Order list, ou seja, lista ordenada, ela te da uma ordem dos itens que são adicionados nela, e a UL que significa Unother List, ou lista desordenada, são separadas nativamente por uma bolinha que pode ser alterada no css posteriormente, exemplos a seguir:

Aqui temos uma UL

  • Lista desordenada - item 1
  • Lista desordenada - item 2
  • Lista desordenada - item 3

Aqui temos uma OL

  1. Lista ordenada - item 1
  2. Lista ordenada - item 2
  3. Lista ordenada - item 3
